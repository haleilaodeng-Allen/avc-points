数据打包工具

把清洗好的酒店 json 打包成可上传的文件夹(含 index.json 清单),缺名字的自动用文件名补上

1 选择所有酒店 json 文件
⬆
点这里多选,或把一堆 .json 拖进来
可一次选中全部 30 个文件(Ctrl+A 全选)

这些是你用清洗工具生成的酒店数据文件。文件名建议是酒店英文名,缺名字的会自动用文件名补。

下一步:上传到 GitHub

  1. 解压下载的 zip,得到一个 hotels 文件夹(里面是所有酒店 json + 一个 index.json)
  2. 把整个 hotels 文件夹上传到你的 GitHub 仓库
  3. 查询系统里填这个文件夹的地址(到 hotels/ 为止),就能自动加载全部酒店

以后加新酒店:把新 json 放进 hotels 文件夹,并在 index.json 里加一行文件名即可。或重新用本工具打包。
